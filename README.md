「AVA-ON」
オンラインFPS『Alliance of Valiant Arms (AVA)』のアカウント管理Chrome拡張機能


【AVA-ON v0.1.2 取扱説明書】

本ドキュメントは、Chrome拡張機能「AVA-ON」の詳細な使い方や各機能の説明、注意点について記載しています。

--------------------------
■ 拡張機能の導入方法
--------------------------
1. ZIPファイルを解凍し、任意の場所に保存します。
2. Chromeの右上の「︙」から「拡張機能」＞「拡張機能を管理」を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍した「AVA-ON」のフォルダを選択します。

--------------------------
■ 各機能の使い方
--------------------------

【1】 アカウント管理 (ACCOUNTS)
複数のAVAアカウントを登録し、簡単に切り替えや管理を行うことができます。

・新規追加 (＋ 新規)
　AVA-ON管理画面（ポップアップ）を開いた状態で「新規」ボタンを押すと、現在のログイン情報がリセットされログイン画面へ移動します。新しくログインするとアカウントが自動的に追加されます。

・アカウント切替 (切替)
　リストからアカウントをクリックして選択し、「切替」ボタンを押すと、そのアカウントにログインした状態になります。

・アカウント順序変更 (▲ / ▼)
　選択したアカウントをリストの上、または下に移動して並び替えることができます。

・アカウント削除 (削除)
　選択したアカウントをリストから削除します。（※不要なトラブルを防ぐため、事前にAVA公式サイトから手動でログアウトしておくことをお勧めします）

・メモ機能 (📝 メモ)
　アカウントを選択し「メモ」ボタンを押すと、そのアカウントに短いメモを登録できます。登録したメモは、アカウント名にマウスを重ねる（ホバーする）ことでツールチップとして表示されます。

・更新 (🔄 更新)
　全アカウントを順番にAVA公式サイトに自動アクセスし、最新のプレーヤー名・ユーロ情報を取得します。
　処理中はブラウザを操作せずにお待ちください。全アカウントの巡回が完了すると元のアカウントに自動復帰し、結果が画面に表示されます。



【2】 一括機能 (AUTOMATION)
登録済みのすべてのアカウントに対して、特定の作業を連続実行する機能です。※処理中は「停止」ボタンで中断可能です。

・ログインキャンペーン (🔄 ログインキャンペーン)
　すべてのアカウントを順番に切り替え、ログインキャンペーンページの報酬を受け取ります。
 　※サイト内の報酬受取作業を簡単にするものでログイン機能ではありません。
　　報酬を受け取るには事前にゲームにログインしておく必要があります。

・アイテム配布 (📥 アイテム配布)
　プロンプトに入力した特定の「アイテム受取用URL」に対し、全アカウントで順番にアクセスしてアイテムを受け取ります。
　※途中で出現する「受け取りました」等のポップアップ通知（アラート）は自動的に閉じられます。

・BOX連続プレイ (🎰 BOX連続プレイ)
　AVA公式サイトの「AVABOXプレイ画面（box_choice）」を開いている時のみ表示される機能です。
　指定した回数分、自動でBOXを連続プレイします。

【3】 クイックナビゲーション (QUICK NAVI)
各ページへのショートカットです。ボタンを押すと現在開いているタブがそのページに推移します。

・HP：AVA公式サイト トップページ
・ITEM：アイテム受け取りページ
・Box：AVABOXトップページ
・⚗️ 合成：合成ページ
・✨ 合成+：合成+ページ
・🧪 調合：調合ページ


--------------------------
■ 注意事項・免責事項
--------------------------
1. 動作環境
・本拡張機能は、PC版のGoogle Chromeでの動作を前提としています。
・AVA公式サイト（ava.pmang.jp または service.pmang.jp）のページを開いている状態でのみ、正常に動作します。公式サイト以外で拡張機能を開くとエラー表示が出ます。

2. 一括機能の利用について
・ログインキャンペーンやBOX連続プレイ等の自動機能は、ネットワーク環境やサーバーの応答速度によって正常に動作しない場合があります。
・実行中は、余計な操作を行わず画面を開いたままお待ちください。連続処理が中断・失敗する原因となります。
・アイテム受取機能は内部でデバッガー機能を使用するため、ブラウザ上部に警告が表示されることがありますが仕様です。
